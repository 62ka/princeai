# Coding AI - StarCoder Edition

A personal, unrestricted coding AI based on **StarCoder**.  
Generates Python, JavaScript, C++, or other code based on prompts and saves it automatically.

---

## Features

- Generates code for websites, games, programs, and more.
- Fully open-source and unfiltered.
- Saves all generated code in a dedicated folder.
- Works locally (no internet required after model download, if cached).

---

## Requirements

- Python 3.10+
- PyTorch
- Transformers library
- GPU recommended (16GB+ VRAM) for best performance, but CPU works too (slower).
